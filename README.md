# CHIP8
A simple CHIP8 emulator in C++ ( std17 ) and SFML

# Folders content
- SRC ( source code )
- SFML ( include, lib, dll )
- RES ( sound file )
- ROMS ( some Chip8 games )

## Usage
- Esc : quit
- F1 : reset
- F2 : load next rom
